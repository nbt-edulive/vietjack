**Bài 2. Trang 59 Vở Bài Tập Toán lớp 2 Tập 2:** Viết số thành tổng các trăm, chục, đơn vị.

a) 514 = ……………………………………………………………………………

b) 920 = ……………………………………………………………………………

c) 407 = ……………………………………………………………………………

d) 63 = ……………………………………………………………………………..

**Trả lời:**

a) 514 = 500 + 10 + 4

b) 920 = 900 + 20 + 0

c) 407 = 400 + 0 + 7

d) 63 = 60 + 3
