____**Phương pháp giải**

\- Tính giá trị ở vế phải. 

\- x đóng vai trò thừa số chưa biết. Muốn tìm thừa số chưa biết ta lấy tích chia cho thừa số đã biết.

**Lời giải:**

a) x × 1,4 = 2,8 × 1,5

x × 1,4 = 4,2

x = 4,2 : 1,4

x = 3 

b) 1,02 × x = 3,57 × 3,06

1,02 × x = 10,9242

x = 10,9242 : 1,02

x = 10,71

[**Vở bài tập Toán lớp 5 trang 87 Bài 3:** Một mảnh vườn hình chữ nhật có diện tích 161,5m2, chiều rộng 9,5m. Tính chu vi mảnh đất hình chữ nhật đó.](https://vietjack.com/giai-vo-bai-tap-toan-5/bai-3-trang-87-vbt-toan-5-tap-1.jsp)

Quảng cáo

____**Phương pháp giải**

Áp dụng các công thức :

\- Chiều dài = diện tích : chiều rộng.

\- Chu vi hình chữ nhật = (chiều dài + chiều rộng) × 2.

**Lời giải:**

Chiều dài mảnh đất hình chữ nhật:

161,5 : 9,5 = 17 (m)

Chu vi mảnh đất hình chữ nhật:

(17 + 9,5) × 2 = 53 (m)

Đáp số: 53m
