____**Phương pháp giải**

Sử dụng tính chất giao hoán và tính chất kết hợp để ghép thành các cặp mà tổng của các số đó là số tự nhiên.

**Lời giải:**

a) 6,9 + 8,75 + 3,1 

= (6,9 + 3,1) + 8,75

= 10 + 8,75 = 18,75

b) 4,67 + 5,88 + 3,12 

= 4,67 + (5,88 + 3,12)

= 4,67 + 9 = 13,67

c) 0,75 + 1,19 + 2,25 + 0,81 

= ( 0,75 + 2,25) + (1,19 + 0,81)

= 3 + 2 = 5
