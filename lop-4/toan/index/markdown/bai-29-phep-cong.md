____**Phương pháp giải**

Áp dụng các quy tắc: Muốn tìm số bị trừ ta lấy hiệu cộng với số trừ. 

**Lời giải:**

a) x – 425 = 625 

x = 625 + 425 

x = 1050 

b) x – 103 = 99

x = 99 + 103

x = 202
