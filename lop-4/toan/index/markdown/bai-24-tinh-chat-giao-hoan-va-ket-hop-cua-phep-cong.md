**Vở bài tập Toán lớp 4 trang 85 Bài 3:** Tính giá trị của biểu thức a + b + c + d với:

a) a = 400; b = 999; c = 600; d = 1. 

a + b + c + d = …………………….

= …………………….

= …………………….

b) a = 75; b = 80; c = 20; d = 25. 

a + b + c + d = …………………….

= …………………….

= …………………….

**Lời giải**

a) a = 400; b = 999; c = 600; d = 1. 

a + b + c + d = 400 + 999 + 600 + 1

= (400 + 600) + (999 + 1)

= 1 000 + 1 000 

= 2 000

b) a = 75; b = 80; c = 20; d = 25. 

a + b + c + d = 75 + 80 + 20 + 25

= (75 + 25) + (80 + 20)

= 100 + 100

= 200
