a. √2 + √3 và √10

Vậy 5 + 2√6 < 5 + 5 ⇒ (√2 + √3 )2 < (√10 )2 ⇒ √2 + √3 < √10 

b. √3 + 2 và √2 + √6

(√2 + √6 )2 = 2 + 2√12 + 6 = 8 + 2√(4.3) = 8 + 2.√4 .√3 = 8 + 4√3 

Vì 7 + 4√3 < 8 + 4√3 nên (√3 + 2)2 < (√2 + √6 )2

Vậy √3 + 2 < √2 + √6
